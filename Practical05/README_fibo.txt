This source code corresponds to a program that computes the nth term in the 
Fibonacci sequence. The only input for the program is a positive integer n. 
For large terms in the sequence whereby they may cause arithmetic overflow, the 
program will return 0 and the message stating arithmetic overflow has occured.

---

In order to compile this source code please use the following command.

fibo -o fibo.c -lm

---

The scipt upon execution will ask the user to enter a positive integer 'n'. 
The program will then print to screen in a vertical fashion the first n terms
of the fibonacci sequence.
